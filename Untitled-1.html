Definition: Cryptography is associated with the process of converting ordinary plain text into unintelligible
text and vice-versa. It is a method of storing and transmitting data in a particular form so that only those
for whom it is intended can read and process it. Cryptography not only protects data from theft or alteration,
but can also be used for user authentication.
In computer science, cryptography refers to secure information and communication techniques
derived from mathematical concepts and a set of rule-based calculations called
algorithms, to transform messages in ways that are hard to decipher. These deterministic
algorithms are used for cryptographic key generation, digital signing, verification to protect
data privacy, web browsing on the internet, and confidential communications such as credit
card transactions and email.
